# 机器智能的数学现象学

> **Mathematical Phenomenology of Machine Intelligence**

本书系统性地梳理了自1959年"机器学习"概念诞生以来，至2026年前沿大模型时代的**全部**核心知识体系。全书共**26章、1560节**，从符号主义与世界模型出发，经由数学基础、经典统计学习、神经网络革命、深度学习浪潮，直至生成式AI与具身智能的最新前沿，构建了一部完整的机器智能百科全书。

---

## 全书结构

| 阶段 | 章节 | 主题 |
|------|------|------|
| **符号智能** | 第0章 | 符号主义、知识推理、元学习、世界模型、神经符号融合 |
| **数学基石** | 第1–4章 | 线性代数、概率统计、最优化、信息论 |
| **经典学习** | 第5–8章 | 统计学习、核方法、图模型、集成学习 |
| **神经网络** | 第9–12章 | 前馈网络、卷积网络、循环网络、注意力机制 |
| **深度学习** | 第13–16章 | Transformer、预训练范式、生成模型、扩散模型 |
| **强化决策** | 第17–18章 | 强化学习基础、深度强化学习 |
| **专题领域** | 第19–22章 | 图神经网络、多模态学习、科学计算、时间序列 |
| **系统工程** | 第23–24章 | 大模型系统工程、AI安全与对齐 |
| **前沿展望** | 第25章 | 2024–2026前沿与未来方向 |

---

## 阅读指南

- **初学者**：建议从第0章了解AI两大范式，再从第1章开始顺序阅读
- **有基础读者**：可直接跳至第5章开始经典机器学习
- **深度学习方向**：第9–16章为核心内容
- **研究人员**：第17–25章涵盖前沿专题
